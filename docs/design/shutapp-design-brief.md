# 셧앱 (ShutApp) — 앱 아이콘 & 스토어 스크린샷 디자인 브리프

> 이 문서는 디자인 AI(예: Claude Design)에게 **앱 아이콘**과 **플레이스토어 스크린샷**을
> 만들어 달라고 요청하기 위한 자기완결형 사양서입니다. 이 문서만 보고 작업할 수 있도록
> 제품 맥락·톤·색상·각 화면의 내용을 모두 담았습니다.

---

## 1. 제품 한 줄 요약

**셧앱(ShutApp)** 은 사용자가 정한 **앱별 하루 사용 시간 한도**를 넘기면, 그 앱 위에
**전체 화면 차단 오버레이**를 띄워 더 못 쓰게 막아주는 Android 디지털 웰빙 앱입니다.

- **카테고리**: 디지털 웰빙 / 집중 / 스크린타임 차단
- **핵심 약속**: "정한 시간을 넘기면, 진짜로 막아준다."
- **표시명**: 한국어 단말 = **셧앱**, 그 외 = **ShutApp** (둘 다 공식 명칭)

---

## 2. 브랜드 아이덴티티

### 포지셔닝 / 톤
- **단호한 차단·통제**가 핵심 정서. "막아준다 · 끊어준다."
- 죄책감을 주는 톤이 아니라 **든든하게 지켜주는 가드(guard)** 의 느낌.
  강하지만 위협적이지 않게.
- 키워드: 절제, 차단, 집중, 멈춤(Stop/Shut), 든든함.

### 네이밍 표기 규칙
- 영문 로고는 **ShutApp** (카멜표기, 대문자 S·A). `shutapp`(전부 소문자) 금지.
- 한글 로고는 **셧앱**.

### 제안 색상 팔레트 (브랜드용 — 이 값을 기준으로 사용)
현재 앱 내부 테마는 Compose 기본 보라색(동적 색상)이라 브랜드 색이 없습니다.
아이콘/스토어 비주얼은 아래 **전용 팔레트**로 통일해 주세요.

| 역할 | 색상 | HEX |
|---|---|---|
| Primary (딥 인디고) | 신뢰·집중의 짙은 남보라 | `#2B2D6E` |
| Primary Dark (배경) | 더 어두운 톤, 그라데이션 하단 | `#14152E` |
| Accent (스톱 레드) | 차단·정지 강조 포인트 | `#FF4D4F` |
| Surface | 카드/패널 밝은 면 | `#F5F6FA` |
| On-Dark Text | 어두운 배경 위 텍스트 | `#FFFFFF` |
| Muted Text | 보조 설명 | `#A0A3BD` |

- **메인 분위기**: 짙은 남보라 그라데이션 배경 + 스톱 레드 포인트.
- 레드는 "정지/차단"을 상징하는 **포인트로만** (전체를 빨갛게 칠하지 말 것).

### 타이포그래피
- 산세리프, 기하학적이고 굵은 굵기(Bold/ExtraBold) 위주.
- 한글: Pretendard / 노토산스 KR 계열 느낌. 영문: Inter / Poppins 계열 느낌.

## 3. 앱 아이콘 사양

### 컨셉 방향 (택1 또는 조합, 디자이너 재량)
1. **Shut = 멈춤/차단 심볼**: 정지(■)나 전원(⏻) 또는 손바닥(✋) 모티프를
   앱 그리드/말풍선과 결합. "앱을 멈춘다"가 직관적으로 읽혀야 함.
2. **차단 게이트**: 둥근 사각형 안에 앱 아이콘이 빗금/차단선(🚫)으로 막힌 형태.
3. **타임 락**: 시계/모래시계 + 자물쇠. 시간이 잠긴다는 의미.

> 1순위 추천: **컨셉 1 (정지·전원 심볼 결합)**. 단순하고 작은 크기에서도 또렷함.

### 필수 요건
- **심볼 1개로 즉시 인식** — 작은 런처 크기(48dp)에서도 뭉개지지 않게 단순하게.
- 배경: 제안 팔레트의 **딥 인디고 그라데이션**(`#2B2D6E` → `#14152E`).
- 포인트: **스톱 레드(`#FF4D4F`)** 를 심볼 일부에 사용.
- 글자(텍스트) 넣지 말 것 — 심볼만. (스토어 썸네일에서 텍스트는 가독성 떨어짐)
- 플랫 / 약한 그라데이션. 사실적 스큐어모피즘·과한 그림자 금지.

### 산출물 (Android)
- **적응형 아이콘(Adaptive Icon)** 기준으로:
  - `foreground` 레이어 (심볼) — 512×512 PNG, 투명 배경, 안전영역 고려
    (가장자리 약 18% 여백, 가운데 66% 영역 안에 핵심 심볼).
  - `background` 레이어 (그라데이션 면) — 512×512 PNG.
- **단일 합본 아이콘** — 1024×1024 PNG (마스킹 없이 전체 보이는 버전).
- **플레이스토어 등록 아이콘** — 512×512 PNG, 32bit, 둥근 모서리 적용 전 정사각.
- 가능하면 **원본 SVG/벡터** 도 함께.

### Do / Don't
- ✅ 단순한 실루엣, 강한 대비, 한 가지 포인트 컬러.
- ❌ 미세한 디테일·얇은 선·작은 텍스트·여러 색 남발.

## 4. 플레이스토어 스크린샷 사양

### 공통 규격
- **휴대폰 세로 스크린샷**, 비율 9:16, 권장 해상도 **1080 × 1920 px** (또는 1242×2208).
- 총 **5장** 제작. 순서대로 스토리가 이어지게.
- 각 장: 상단 **헤드라인(한 줄 카피)** + 그 아래 **기기 목업** 안에 해당 앱 화면.
- 배경은 브랜드 딥 인디고 그라데이션, 카피는 흰색 Bold + 핵심 단어만 스톱 레드 강조.
- 카피는 **한국어** 기준 (영문판 필요 시 병기 버전 별도 요청).

### 스크린샷별 내용 (앱 실제 화면 기준)

> 아래 "화면 내용"은 실제 앱 UI를 그대로 묘사한 것입니다. 목업 안 화면은 이 설명대로
> 그려 주세요. Material3 디자인, 깔끔한 카드형 리스트.

**① 홈 — 한도 현황 한눈에**
- 헤드라인: **"오늘, 얼마나 썼는지 한눈에"**
- 화면 내용: 상단 타이틀 "셧앱". 아래로 앱별 카드 리스트. 각 카드 =
  앱 아이콘 + 앱 이름 + **사용량/한도**(예: `48분 / 60분`) + 진행 막대(progress bar).
  한도 근접한 카드는 막대가 빨강(`#FF4D4F`)에 가까움. 우하단 **+ 플로팅 버튼**.
  맨 위에 "주말 동안 제한 일시정지" 버튼 1개.

**② 추가 — 앱 고르고 시간 정하기**
- 헤드라인: **"막고 싶은 앱, 시간만 정하면 끝"**
- 화면 내용: "앱 선택" 검색창 + 설치 앱 리스트(아이콘+이름). 아래 "한도 설정"
  슬라이더(5분 ~ 4시간) + 현재 값 표시. 하단 **저장** 버튼.

**③ 차단 오버레이 — 핵심 기능 (가장 강조)**
- 헤드라인: **"넘기면, 진짜로 막아줍니다"** ("막아" 를 스톱 레드 강조)
- 화면 내용: 어두운 전체 화면 오버레이. 중앙에 **"오늘의 사용 한도에 도달했습니다"**,
  그 아래 `오늘 60 / 60 분 사용`, **자정까지 남은 카운트다운**(예: `06:12:33 후 초기화`),
  "셧앱 열기" 버튼, 그 아래 흐릿한 "15분 더 허용"(스누즈) 버튼. 우상단 ✕ 닫기.
  하단에 작은 배너 영역. → **차단당한 앱이 가려진 느낌**이 들도록 연출.

**④ 알림 — 미리 경고해줌**
- 헤드라인: **"한도 가까워지면 미리 알려줘요"**
- 화면 내용: 상태바 알림 목업. 제목 `(앱이름) – 시간 한도가 얼마 남지 않았습니다`,
  내용 `오늘 12분 남았습니다`. 80% 도달 시 하루 한 번 오는 경고 알림.

**⑤ 유연함 — 주말엔 잠깐 쉬어가기**
- 헤드라인: **"주말·휴가엔 일시정지"**
- 화면 내용: 홈 화면에서 "주말 동안 제한 일시정지" 버튼이 활성화되어
  "제한 재개"로 바뀐 상태. 다음 월요일 자정까지 모든 차단이 멈춘다는 안내.

### 스크린샷 Do / Don't
- ✅ 장마다 헤드라인 1줄 + 화면 1개. 일관된 배경/폰트/목업 프레임.
- ✅ ③ 차단 화면을 시리즈의 중심(가장 임팩트 있게).
- ❌ 한 장에 화면 여러 개 쑤셔넣기, 장마다 다른 스타일, 실제 없는 기능 묘사.

## 5. 참고 — 앱이 보유한 실제 기능 (허구 금지 체크리스트)

디자인에 등장시켜도 되는 **실재하는** 기능만 정리합니다. 이 목록에 없는 기능
(예: 통계 그래프, 친구 랭킹, 로그인)은 만들어 넣지 마세요.

- 앱별 **하루 사용 시간 한도** 설정 (5분~4시간 슬라이더)
- 한도 초과 시 **전체 화면 차단 오버레이** + 자정까지 카운트다운
- **스누즈**: "15분 더 허용" (한도 +15분)
- 80% 도달 시 **경고 알림**, 100% 도달 시 차단
- **주말/휴가 일시정지** (다음 월요일 자정까지 전체 차단 해제)
- 사용량 **수동 초기화(↺)** 및 **당겨서 새로고침**
- 포그라운드 서비스로 백그라운드 모니터링 (상시 알림 1개)
- 하단 **배너 광고** / 일부 동작 시 **전면 광고** (AdMob)
