# 셧앱 (ShutApp) — 스토어 그래픽 이미지(Feature Graphic) 디자인 브리프

> 이 문서는 디자인 AI(예: Claude Design, 이미지 생성 모델)에게 **구글 플레이스토어 그래픽
> 이미지(Feature Graphic / 피처 그래픽)** 를 만들어 달라고 요청하기 위한 자기완결형 사양서입니다.
> 앱 아이콘·스크린샷 사양은 같은 폴더의 [`shutapp-design-brief.md`](./shutapp-design-brief.md)
> 를 참고하세요. 색상·톤·네이밍 규칙은 그 문서와 **반드시 일치**시켜야 합니다.

---

## 0. 그래픽 이미지란

구글 플레이 스토어 상세 페이지 **최상단(앱 이름·아이콘 위)** 에 가로로 크게 노출되는
배너 이미지입니다. 검색 결과·추천·프로모션 영역에서도 쓰입니다. 첫인상을 좌우하는 핵심 비주얼.

---

## 1. 필수 규격 (Google Play)

| 항목 | 값 |
|---|---|
| 크기 | **1024 × 500 px** (고정, 가로형) |
| 포맷 | PNG(권장, 24bit, 투명도 없음) 또는 JPG |
| 최대 용량 | 15 MB |
| 텍스트 | 넣어도 되지만 **최소한으로** (영역 일부가 UI에 가려질 수 있음) |
| 안전 영역 | 가장자리에서 안쪽으로 약 **5%(상하좌우 ≈ 50px / 25px)** 여백 확보. 핵심 요소·로고·카피는 **중앙 안전 박스(약 924×400px)** 안에 배치 |
| 모서리/오버레이 | 스토어가 자동으로 모서리를 둥글게 깎고, 위에 재생 아이콘·텍스트가 겹칠 수 있음 → **모서리에 중요한 것 두지 말 것** |

---

## 2. 브랜드 (요약 — 상세는 design-brief §2)

- **표시명**: 한글 = **셧앱**, 영문 = **ShutApp** (카멜표기, 대문자 S·A). `shutapp` 금지.
- **핵심 약속**: "정한 시간을 넘기면, 진짜로 막아준다."
- **톤**: 단호한 차단·통제 + 든든한 가드. 강하지만 위협적이지 않게.

### 색상 팔레트 (이 값 그대로 사용)

| 역할 | HEX |
|---|---|
| Primary (딥 인디고) | `#2B2D6E` |
| Primary Dark (그라데이션 하단) | `#14152E` |
| Accent (스톱 레드) | `#FF4D4F` |
| Accent 보조 (그라데이션 끝) | `#FF785A` |
| On-Dark Text | `#FFFFFF` |
| Muted Text | `#A0A3BD` |

- 배경: **딥 인디고 → 더 어두운 남보라 대각선 그라데이션**(`#2B2D6E` → `#14152E`).
- 레드는 **포인트로만** (전원/정지 심볼, 핵심 단어 강조). 전체를 빨갛게 칠하지 말 것.
- 앱 아이콘 안의 빨간 전원(⏻) 심볼과 톤을 맞출 것.

---

## 3. 레이아웃 (택1 — 1순위 추천: A)

### A. 로고 + 심볼 + 한 줄 카피  ⭐ 추천
```
┌──────────────────────────────────────────────────────────┐
│   (딥 인디고 그라데이션 배경 + 은은한 빛 번짐)              │
│                                                            │
│     ⏻   셧앱 ShutApp                                       │
│     │                                                      │
│     └  넘기면, 진짜로 막아줍니다                            │
│         (작은 보조: 앱별 하루 사용시간 제한)                │
│                                                            │
│                                  [폰 목업 — 차단화면, 우측] │
└──────────────────────────────────────────────────────────┘
```
- **좌측**: 빨간 전원(⏻) 심볼 + "셧앱 ShutApp" 로고(흰색 Bold).
- 그 아래 헤드라인 **"넘기면, 진짜로 막아줍니다"** — "막아"만 스톱 레드.
- **우측**: 차단 오버레이 화면이 담긴 **폰 목업**(살짝 기울여 떠 있는 느낌, 그림자).
- 카피는 왼쪽 정렬, 폰은 오른쪽. 시선이 좌→우로 흐르게.

### B. 심볼 중심 (텍스트 최소)
- 중앙에 큰 빨간 전원/정지 심볼 + 그 아래 "셧앱 ShutApp"만. 카피 없음.
- 다국어 노출·미니멀 선호 시. 작은 썸네일에서도 안전.

### C. 비포/애프터 대비
- 좌: 알림으로 뒤덮인 산만한 폰 → 우: 차단 화면으로 정돈된 폰.
- 가운데 화살표/구분선. "끊어준다"를 서사로 보여줌. (제작 난이도 ↑)

---

## 4. 카피 (한국어 / 영어)

그래픽 이미지에 들어갈 짧은 카피. **한 장에 헤드라인 1줄**만 권장.

| 용도 | 한국어 | English |
|---|---|---|
| 메인 헤드라인(추천) | 넘기면, 진짜로 막아줍니다 | When time's up, it really stops you |
| 대안 1 | 앱 사용시간, 내가 정한 만큼만 | Use apps only as much as you decide |
| 대안 2 | 스마트폰 그만, 셧! | Time to shut it down |
| 보조(서브) 카피 | 앱별 하루 사용시간 제한 | Daily limits for every app |

- 강조 단어(레드): 한국어 "**막아**" / 영어 "**stops**".
- 영문판 그래픽은 로고를 "ShutApp" 단독으로.

## 5. 이미지 생성 프롬프트 (그대로 복붙용)

### 한국어 (레이아웃 A 기준)
```
가로 1024×500 배너. 딥 인디고에서 짙은 남보라로 이어지는 대각선 그라데이션 배경
(#2B2D6E → #14152E), 은은한 빛 번짐. 왼쪽에 둥근 사각 안의 빨간(#FF4D4F) 전원 버튼
심볼과 흰색 볼드 로고 "셧앱 ShutApp", 그 아래 흰색 헤드라인 "넘기면, 진짜로 막아줍니다"
("막아"만 빨강 강조), 작은 회색(#A0A3BD) 보조문구 "앱별 하루 사용시간 제한".
오른쪽에는 약간 기울어 떠 있는 스마트폰 목업, 화면에는 어두운 차단 오버레이와
큰 카운트다운 타이머와 빨간 둥근 버튼. 플랫/미니멀, 기하학적 산세리프, 강한 대비,
사실적 그림자 절제. 모서리와 가장자리에는 중요한 요소 배치 금지(안전 여백 확보).
```

### English (layout A)
```
A 1024×500 horizontal banner. Diagonal gradient background from deep indigo to dark
navy-purple (#2B2D6E → #14152E) with subtle glow. On the left: a red (#FF4D4F) power
button symbol inside a rounded square, white bold logo "ShutApp", below it a white
headline "When time's up, it really stops you" (the word "stops" in red), and a small
muted (#A0A3BD) subline "Daily limits for every app". On the right: a slightly tilted
floating smartphone mockup showing a dark blocking overlay with a large countdown timer
and a red rounded button. Flat, minimal, geometric sans-serif, high contrast, restrained
realistic shadow. Keep important elements away from corners/edges (safe margins).
```

> 목업 안 차단화면은 실제 앱 화면을 따르세요: 어두운 남보라 배경, 앱 아이콘, "오늘의 사용
> 한도에 도달했습니다", 큰 카운트다운(예: `05:42:18`) + "후 자정에 초기화", 빨강
> 그라데이션 "셧앱 열기" 버튼, 흐릿한 "15분 더 허용" 버튼, 하단 배너 영역.

## 6. Do / Don't

- ✅ 배경 그라데이션 + 한 가지 포인트 컬러(스톱 레드). design-brief 팔레트 준수.
- ✅ 카피는 1줄, 로고는 또렷하게. 안전 여백 확보.
- ✅ 폰 목업 안 화면은 §5 주석의 **실재하는 차단 화면**만.
- ❌ 가장자리/모서리에 로고·카피 배치(잘림·오버레이 위험).
- ❌ 전체를 빨갛게, 텍스트 빽빽하게, 여러 폰트·색 남발.
- ❌ 없는 기능(통계 그래프·랭킹·로그인 등) 묘사 — design-brief §5 체크리스트 준수.
